======功能介绍======  
没啥可介绍的 它会自己算出挖宝开箱子吃了多少系统金然后发出来

======食用方法======  
- 导入并设置鲶鱼精邮差(参考版内鲶鱼精邮差相关内容)  
- ACT高级触发器导入即可(参考版内高级触发器相关内容)  
- 请保证基础功能全部勾选  
- 可选功能按实际情况酌情开启

1. 开始挖宝时的金币重置(建议写在宏里并放在**不**经常按的位置):  
    ps.只有老板才需要的功能 如果打工的话在进招募的时候会自动重置   <font color=ff0000>[重置操作不可逆!操作需谨慎]  </font>

```/e 挖宝查询```

```/p 挖宝查询```
```/p 挖宝统计```
